**Прогнозирование цен на криптовалюту с использованием анализа новостного фона и Deep Learning.**

Этот репозиторий содержит код и материалы исследовательско-практической работы, посвященной применению методов глубокого обучения (Deep Learning) и обработки естественного языка (NLP) для алгоритмической торговли. Основная идея — переход от прямой регрессии цен к анализу информационного фона и настроений рынка (Sentiment Analysis).

### О проекте

Цель работы — выяснить, способны ли модели ML и DL улучшить прогнозирование цен на высоковалотильном рынке криптовалют через оценку новостного фона.
Ключевая гипотеза заключается в том, что использование сентимент-анализа новостей (позитивный/негативный/нейтральный) в сочетании с техническими индикаторами позволяет более точно предсказывать направление тренда и управлять рисками.

### ⚙️ Архитектура и Пайплайн

Проект состоит из трех основных этапов:

1. **Сбор данных (Parsing):** Скрапинг новостей с CoinDesk и исторических данных OHLCV с биржи Kucoin.
2. **Анализ тональности (Sentiment Analysis):** Обработка текстов с помощью **FinBERT** для получения оценки настроения рынка.
3. **Прогнозирование тренда (Trading Strategy):** Агрегация данных и использование **Random Forest** для генерации торговых сигналов (Buy/Sell/Hold).

### Данные

* 
**Новости:** Собрано более 180,000 статей с ресурса CoinDesk.


* 
*Особенность:* Использован принцип «перевернутой пирамиды» — для анализа берутся только первые 256 токенов, так как основная смысловая нагрузка содержится в заголовке и первом абзаце. Это позволило увеличить обучающую выборку до 85,000 примеров и избежать проблем с памятью (OOM).




* 
**Рыночные данные:** Часовые свечи (Open, High, Low, Close, Volume) за период более 2 лет (2023–2025).



### Модели

#### 1. Sentiment Analysis: FinBERT (Fine-tuned)

В качестве основного инструмента использована модель **FinBERT**, дообученная на собранном датасете криптовалютных новостей.

* 
**Сравнение:** FinBERT показал лучшую масштабируемость и понимание контекста (особенно отрицаний) по сравнению с LSTM и базовым BERT.


* 
**Точность:** После дообучения (3 эпохи) точность (Accuracy) на валидации достигла **~82%**.



#### 2. Trading Model: Random Forest

Для принятия торговых решений используется ансамбль решающих деревьев (Random Forest), реализующий механику «мудрости толпы».
**Признаки (Features) для модели:**

* Агрегированный сентимент за 1ч, 4ч, 12ч, 24ч, 168ч.


* Технические индикаторы: **RSI** (индекс относительной силы) и **ATR** (индикатор волатильности).


* Процентные изменения цены и объема.



### Торговая стратегия

* 
**Сигналы:** Модель предсказывает движение цены (1 - Long, -1 - Short, 0 - Hold).


* 
**Риск-менеджмент:** Соотношение Risk:Reward (R:R) установлено как **1:2**.


* 
**Take-Profit / Stop-Loss:** Рассчитываются динамически на основе ATR.


* 
**Порог уверенности:** Сделки открываются только при уверенности модели выше **0.52**.
